Provides simulation algorithms for cellular automatons.

General description

This package provides simulation algorithms for cellular automatons realized using the "rule based" cellular automata formalism. the algorithm here are provided as examples how different simulation algorithm for a single formalism can be realized, and can coexist in the framework.
