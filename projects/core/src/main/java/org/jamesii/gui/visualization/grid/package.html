Provides a flexible grid component to visualize grids.

Some formalisms, e.g., cellular automata are grid based formalisms, and thus, if they shall be visualized, a grid visualization is required. In addition grids are often used as an abstraction level for geographical data, and thus run time visualizations with per grid cell information are in need of such a visualization as well.
Besides visualization this grid implementation can be used to edit grids as well, e.g., to set initial states.

The grid component provides support for

  • 1-D grids
  • 2-D grids

In accordance to other visual components data to be displayed is hold in a special model.
