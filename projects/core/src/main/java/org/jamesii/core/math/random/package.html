The random number package.

General description

Many modelling formalisms and simulation algorithms depend on stochasticity, and thus on pseudo random number generation. This packages contains the basic interfaces for random number in the modelling and simulation framework. In modelling and simulation the full control over "random" processes is mandatory: the repeatability of experiments has to be guaranteed.
The sub package generators contains the extension point for pseudo random number generators.
The sub package seedgenerators contains the extension point for seed generators for initializing pseudo random number generators.
The sub package distributions contains implementations of distributions which are fed by one of the random number generators. The random number generators typically produce equally distributed random numbers, however we often need random number distributed according to a certain distribution, e.g., exponentially distributed numbers.
The RandomSampler
class finally provides methods for sampling random sets.
