The random number distributions package.

General description

In modelling and simulation you often need random numbers distributed according to different distributions. Pseudo random number generators typically produce equally distributed series of numbers which can be modified, so that different distributions can be produced.
This package contains different random distributions, all added as plug-ins to the framework.
Each distribution has to be fed by a pseudo random number generator, which allow the combination of any available random number generator with their implementations.
