Provides means for data input / output handling..

Package use

A M&S software needs a variety of data input / output handling means to be ready for different challenges.
This includes
  • model reading / writing
  • experiment definition reading / writing
  • experiment suite definition reading / writing
  • simulation run trajectories reading / writing
  • resilience information reading / writing
This packages, together with its subpackages provides the basic interfaces for these. Specific plug-ins, extending the framework's functionality, can be added to their corresponding extension points defined in the subpackages for the different read/ write tasks.
