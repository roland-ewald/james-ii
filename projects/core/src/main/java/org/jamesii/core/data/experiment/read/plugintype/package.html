Provides the "experiment definition reader" extension point (plug-in type).

Package use

An extension point allows to provide new functionality to the framework.
Specific plug-ins, extending the framework's functionality, can be added to their corresponding extension points.

An extension points comprises three different files

  • an abstract factory which selects the factory of the plug-in to be used
  • a base factory from which the instance creating factory of each plug-in has to inherit
  • a plug-in type xml based description file named plugintype.xml
