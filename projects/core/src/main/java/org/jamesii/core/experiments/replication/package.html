Provides replication criteria.

Repetitions of single simulation runs are needed whenever there is stochasticity in the models, or if a number of runs shall make sure that side effects will not have a major impact on the final results. Replication criteria are thus a part of an experiment definition {@link james.core.experiments.BaseExperiment}.
