The parse tree / calculation tree package - condition package.

General description

Many modeling formalisms / languages allow to define mathematical expressions. Therefore the framework ships with a default implementation of a parse / calculation / arithmetic tree which supports many of the basic arithmetic operations as well as some operations on lists.

The control package provides nodes representing control statements, like if-then-else or case statements.
