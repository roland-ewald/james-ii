Provides a flexible chart component to visualize simulation trajectories.

On executing simulations often the need arise to visualize the development of some parameters while the simulation is running.
This component is not intended to be a full replacement for any other chart component, or application which could be used to draw charts, but it is intended to be a good choice to visualize simulation runs online without the need of any extra package.

The chart component provides support for

  • line charts
  • multiple axis scales

In accordance to other visual components data to be displayed is hold in a special model.
