Provides the basics for factory based plug-in instance creation.

The plug-in based concept is based on the abstract factory ({@link james.core.factories.AbstractFactory}) and the factory ({@link james.core.factories.Factory}) patterns. To be more concrete: An abstract factory takes care of selecting an appropriate factory for a given task per plug-in type. This selection process is based on a set of criteria which are used to filter / and or sort the remaining factories.
