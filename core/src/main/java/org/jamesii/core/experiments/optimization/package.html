Provides basic optimization means.

Optimization experiments are experiments with models with the purpose of optimizing the model parameters so that the results match a given function. Many applications of modeling and simulation are in fact optimization applications. The framework does support any optimization algorithm (whether sequential or parallel). Here the basic interfaces, classes and extension points are provided to integrate any optimization technique, and to make them usable from within the {@link james.core.experiments.BaseExperiment} class.
