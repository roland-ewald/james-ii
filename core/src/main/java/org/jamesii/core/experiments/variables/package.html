Provides experiment variables.

An experiment typically means a number of simulation runs with different parameter combinations. A {@link james.core.experiments.BaseExperiment} has with its experiment variables attribute the ability to define a set of parameters and the rules how to modify these from one run to the other. This package provides the basic classes for these variables. In the sub package modifier the basics for the realization of the modification rules are laid.
