Provides distributed simulation run, and service / simulation server management for parallel and parallel distributed simulation runs.

Here functionality for the execution of distributed simulations has been realized on base of the "service" notion in the framework. Among those, {@link james.core.simulation.distributed.masterserver.IMasterServer} and {@link james.core.simulation.distributed.simulationserver.ISimulationServer}, partitioning algorithms, and load balancing algorithms extension points.
