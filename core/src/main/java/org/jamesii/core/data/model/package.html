Provides model reading / writing.

General description

The frameworks differentiates between symbolic and executable models. The first ones are comparable to normal "source code", the latter ones to compiled and executable programs. To stay in this metaphor a simulation algorithm then is comparable to the CPU the executable program is executed by.
this package provides the means to work with symbolic model - to read them from data source, and to write them to those, and to compile the symbolic models into executable ones.
