Provides the base classes most classes in the framework are subclasses of.

General description

Herein are classes which implement basic functionality required throughout the m&s framework. Basically these classes extend the object class by very simple additions, however, these are required quite often. On the one site all classes which use the classed in here as base class are observable, and on the other site frequently used attribures are introduced, e.g., a "name" attribute in the {@link james.core.base.INamedEntity} interface and its default implementation the {@link james.core.base.NamedEntity} class.
